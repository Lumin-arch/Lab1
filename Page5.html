Linux קל find — חיפוש קבצים במערכת הקבצים
מאתר קבצים ותיקיות לפי שם, סוג, הרשאות, גודל וזמן שינוי. כלי חיוני לחיפוש SUID ולפי metadata. 
# חיפוש לפי שם
find /home -name "*.conf" -type f

# קבצי SUID (הסלמת הרשאות)
find / -perm -4000 -type f 2>/dev/null

# קבצים שנערכו בשעה האחרונה
find /etc -mmin -60

📖 צלילה עמוקה

אופרטורים נפוצים

-name תיאור רגיש לרישיות · -iname לא רגיש

טיפ: שילוב עם exec

find /tmp -name "*.sh" -exec chmod +x {} \;
Linux קל grep — חיפוש טקסט בקבצים
חיפוש דפוסי טקסט ו-regex בקבצים ובפלט פקודות. בלתי-נפרד מעבודת shell.
# חפש מחרוזת בקובץ (רגיש לרישיות)
grep "error" /var/log/syslog

# רקורסיבי + לא רגיש לרישיות + מספרי שורות
grep -rin "password" /etc/

# regex: הדפס רק את ה-match עצמו
grep -oP '\b\d{1,3}(\.\d{1,3}){3}\b' file.txt

Docker בינוני docker run — הפעלת קונטיינרים
הפעל קונטיינרים עם אפשרויות מפורטות: רשת, volumes, envs וחשיפת פורטים.
# הפעל קונטיינר עם פורט וenv
docker run -d -p 8080:80 -e ENV=prod --name web nginx

# עם volume מקומי
docker run -v /host/data:/app/data myapp

# הרץ ומחק אחרי יציאה
docker run --rm -it ubuntu bash

Cisco בינוני show ip route — טבלת ניתוב
הצג טבלת ניתוב, פרוטוקולים פעילים ומסלולים ספציפיים בנתבי Cisco IOS.
# טבלת ניתוב כוללת
Router# show ip route

# רק מסלולי OSPF
Router# show ip route ospf

# פרטי ממשק
Router# show interfaces GigabitEthernet0/0

# סיכום VLAN
Switch# show vlan brief

Git קל git log — היסטוריית commits
הצגה וסינון של היסטוריית commits עם פורמטים שונים.
# גרף ויזואלי צבעוני
git log --oneline --graph --all

# commits של משתמש בשבועיים האחרונים
git log --author="Avraham" --since="2 weeks ago"

# הצג שינויים בכל commit
git log -p --follow src/app.js

Network מתקדם nmap — סריקת רשת ופורטים
סריקת פורטים וגילוי מכשירים ברשת. כלי יסודי לבדיקות penetration.
# סריקה בסיסית עם גילוי שירותים
nmap -sV -sC 10.0.1.0/24

# כל הפורטים + עצמות הפלט
nmap -p- -T4 -A -oN scan.txt TARGET_IP

# UDP scan (דורש sudo)
sudo nmap -sU --top-ports 100 TARGET_IP

📖 צלילה עמוקה

⚠ שימוש אחראי

הרץ nmap רק על רשתות שיש לך הרשאה לסרוק. סריקה ללא הרשאה היא עבירה חוקית.
